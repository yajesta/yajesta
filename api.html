GET /api/now-playing

Returns the track currently playing on Spotify, or the most recently played track if nothing is active. Absoultely no reason for you to use this, why would you embed what I listen to lol?.

URL
https://oscarkalid.com/api/now-playing
Response
{ "isPlaying": true, "title": "Song Name", "artist": "Artist Name", "albumArt": "https://i.scdn.co/image/...", "songUrl": "https://open.spotify.com/track/..." }
Status Codes
200 Success. Returns track data, or { isPlaying: false } if no data is available.
500 Internal error fetching from Spotify.
